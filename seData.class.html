Класс seData

Основной класс для управления статическими данными сайта.
Построен по принципу singleton, не требует вызова конструктора.

Основы работы с классом

Получение экземпляра объекта производится вызовом статического метода getInstance

$data = seData::getInstance($namepage);

$namepage - название страницы в текущем рабочем проекте.
